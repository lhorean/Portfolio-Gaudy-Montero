# Definición de mi perfil profesional objetivo

Desarrolladora Web con enfoque Full Stack, ya que me interesa comprender y desarrollar tanto la parte visual e interactiva de una aplicación (frontend), como la lógica, bases de datos y arquitectura del sistema (backend).

Me gusta trabajar en ambos entornos y me motiva la posibilidad de entender el funcionamiento integral de una aplicación. Soy consciente de que el camino hacia un perfil Full Stack sólido requiere práctica y aprendizaje continuo, por lo que mi objetivo es fortalecer progresivamente mis habilidades técnicas en ambas áreas para convertirme en una profesional completa y versátil.

# Justificación del Enfoque del Sitio Web

El sitio web desarrollado funciona como un portafolio profesional orientado a la postulación de un puesto Full Stack Junior ficticio, alineado con los perfiles analizados en las cinco ofertas investigadas.

La estructura del sitio refleja:

- Sección de habilidades frontend (HTML, CSS, JavaScript)
- Sección de conocimientos backend (bases de datos, APIs, lógica)
- Proyectos académicos que demuestran aplicación práctica

El diseño y contenido del portafolio están pensados para demostrar:

- Comprensión del desarrollo web moderno
- Interés en el aprendizaje continuo
- Visión integral del desarrollo de software

---

# Diferenciadores Clave

A diferencia de un perfil exclusivamente frontend o backend, mi enfoque Full Stack me permite comprender la conexión entre la experiencia del usuario y la lógica del sistema, facilitando soluciones más completas y coherentes.

Las ofertas analizadas muestran que la actualización constante es clave. Mi objetivo es mantenerme en formación continua, aprendiendo nuevas tecnologías y buenas prácticas para adaptarme a las necesidades del mercado.
